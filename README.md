# 🌙 Somneria - Acceptance Tests

Este repositorio contiene los **Acceptance Tests** del proyecto **Somneria**, desarrollados para el curso de **Ingeniería de Requisitos**.

Las pruebas fueron implementadas utilizando **Behavior-Driven Development (BDD)** y redactadas en lenguaje **Gherkin**, siguiendo la estructura **Feature, Background, Scenario, Scenario Outline y Examples** para validar las funcionalidades priorizadas del sistema.

## EP-005 Organización de Hábitos

Esta épica reúne las funcionalidades relacionadas con la creación, seguimiento y mejora de los hábitos digitales y de descanso del estudiante.

## EP-006 Asesoría y Bienestar Digital
